# Critical Ops Dumps

Dumps for Critical Ops, organized by version (e.g. `1.60.0.f3141`). Join the [Discord](https://discord.gg/q6cra6wUQA) for more.

## Structure
Each folder contains `.rar` archives with `dump.cs` inside.

- **Note:** Starting from version `1.60.0.f3191`, each version folder contains two dumps: `dump-x86-64.rar` and `dump-arm64-v8a.rar`. 
- Versions prior to `1.60.0.f3191` contain x86_64 builds only.
- New dumps added with every game update.
