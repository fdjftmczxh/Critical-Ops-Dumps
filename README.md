# Critical Ops SDK Dumps

This repository serves as a central container for SDK dumps of Critical Ops.

## Repository Structure

Dumps are organized into folders named after the game version (e.g. `1.60.0.f3141`).

Each folder contains the `.rar` archive with `dump.cs` inside.

## Contents

- `dump.cs` – the full SDK dump (C# format)
- All dumps are provided **only** as `.rar` archives due to GitHub's file size limits

## Supported Versions

- Currently targeting **x86_64** builds of the game only
- No ARM64 or other architectures are included at this time

## Usage

1. Navigate to the folder matching your game version
2. Download the `.rar` file
3. Extract `dump.cs`
4. Use the dump as needed for your projects

## Notes

- This repository only stores the raw dumps. No additional documentation, tools, or loaders are provided.
- SDK updates for x86_64 will be added with every new game update.
